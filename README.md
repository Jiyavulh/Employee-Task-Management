# Employee Task Management System

## 📌 Project Description
A web-based Employee Task Management System developed using Spring MVC, Hibernate (JPA), and MySQL.  
This application allows managers to manage employees and assign tasks efficiently.

## 🚀 Features
- Manager Registration & Login
- Add / Update / Delete Employee
- Create Task
- Assign Task to Employee
- View Assigned Tasks
- Change Task Status
- Email Verification
- Password Update

## 🛠 Tech Stack
- Java
- Spring MVC
- Hibernate (JPA)
- MySQL
- JSP / HTML
- Maven
- Tomcat Server

## 🏗 Architecture
- Controller Layer
- Service Layer
- DAO Layer
- Entity Layer
- View Layer (JSP/HTML)

## 💾 How to Run
1. Import project as Maven Project
2. Configure MySQL database
3. Update DB credentials in configuration
4. Run on Apache Tomcat Server
